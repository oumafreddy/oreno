HIGH Missing Content-Security-Policy (CSP)
No CSP header. XSS attacks have no browser-level mitigation.
Fix:
Define a CSP that restricts script sources. Start with:
Content-Security-Policy: default-src 'self'; script-src 'self'; object-src 'none'
Django: use django-csp package.
MEDIUM Missing Strict-Transport-Security (HSTS)
HSTS is not set. Browsers won't enforce HTTPS on repeat visits.
Fix:
Add header: Strict-Transport-Security: max-age=31536000; includeSubDomains; preload
Django: SECURE_HSTS_SECONDS = 31536000
MEDIUM Missing X-Frame-Options (clickjacking protection)
The site can be embedded in iframes, enabling clickjacking attacks.
Fix:
Add header: X-Frame-Options: DENY
Django: X_FRAME_OPTIONS = "DENY"
LOW Missing X-Content-Type-Options: nosniff
Browsers may MIME-sniff responses, enabling content injection attacks.
Fix:
Add header: X-Content-Type-Options: nosniff
Django: SECURE_CONTENT_TYPE_NOSNIFF = True
LOW Missing or weak Referrer-Policy
Full URLs (including paths and query strings) may be leaked to third parties via the Referer header.
Fix:
Add header: Referrer-Policy: strict-origin-when-cross-origin
LOW Missing Permissions-Policy header
No restrictions on browser features (camera, microphone, geolocation). Malicious scripts could abuse them.
Fix:
Add header: Permissions-Policy: geolocation=(), microphone=(), camera=()
